# qdrant-skills

Project-local Cursor skill for Qdrant and vector search work in this repository.

## Purpose
This skill gives the agent a project-specific checklist for retrieval, embeddings, and Qdrant integration work.

## When to use
Use it for tasks involving:
- vector search
- embeddings
- RAG
- Qdrant integration
- retrieval configuration
- deployment changes related to vector storage
