# SmartTech 员工手册

## 第一章 考勤制度

### 1.1 工作时间
- 标准工作时间：周一至周五，9:00-18:00
- 午休时间：12:00-13:30
- 弹性上班：可在8:30-9:30之间打卡，对应下班时间顺延
- 核心工作时间：10:00-16:00（此时间段必须在岗）

### 1.2 迟到与早退
- 迟到30分钟以内：不扣薪，但当月累计超过3次需面谈
- 迟到30分钟以上：按旷工半天处理
- 早退：需提前向主管申请，否则按旷工处理

### 1.3 加班制度
- 工作日加班：需提前在OA系统提交加班申请
- 周末加班：可选择调休或1.5倍工资
- 法定节假日加班：3倍工资
- 每月加班时间不得超过36小时

## 第二章 休假制度

### 2.1 年假
- 入职满1年：5天年假
- 入职满3年：7天年假
- 入职满5年：10天年假
- 入职满10年：15天年假
- 年假可分次使用，最小单位为半天
- 当年未使用的年假可顺延至次年3月31日前使用

### 2.2 病假
- 每年有10天带薪病假
- 超过3天需提供医院证明
- 病假期间工资按80%发放
- 连续病假超过30天，按相关法规处理

### 2.3 事假
- 事假为无薪假期
- 每月事假不超过3天可直接审批
- 超过3天需部门负责人和HR共同审批
- 全年事假累计不超过15天

### 2.4 婚假、产假、陪产假
- 婚假：法定3天 + 公司额外7天 = 10天
- 产假：女性158天，难产增加15天
- 陪产假：男性20天
- 以上假期均为带薪假期

## 第三章 薪酬福利

### 3.1 薪资结构
- 月薪 = 基本工资（70%）+ 绩效工资（20%）+ 补贴（10%）
- 每月15日发放上月工资
- 年终奖：根据公司业绩和个人绩效，通常为1-4个月工资

### 3.2 社保公积金
- 五险一金：按照国家规定比例缴纳
- 公积金：公司缴纳比例为12%（最高标准）
- 补充商业保险：公司额外为员工购买补充医疗保险

### 3.3 其他福利
- 餐补：每月500元
- 交通补：每月300元
- 通讯补：每月200元
- 生日福利：500元购物卡
- 节日福利：春节、中秋等节日发放礼品或购物卡
- 年度体检：每年一次全面体检
- 团建经费：每季度一次部门团建

## 第四章 培训发展

### 4.1 新员工培训
- 入职第一周：公司文化、制度、工具培训
- 入职第一个月：导师1对1辅导
- 试用期3个月内：完成部门专业技能培训

### 4.2 职业发展
- 技术序列：初级工程师 → 中级工程师 → 高级工程师 → 架构师 → 技术总监
- 管理序列：组长 → 主管 → 经理 → 总监 → VP
- 每年两次晋升窗口（4月和10月）
- 晋升需通过技术/管理能力评估

### 4.3 培训支持
- 外部培训：公司报销80%费用（需审批）
- 技术书籍：每季度500元书籍补贴
- 技术认证：通过后公司全额报销考试费
